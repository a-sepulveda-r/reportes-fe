# Frontend

## Requisitos previos

- Node.js instalado en tu máquina.

## Pasos para ejecutar el proyecto

```bash
# Clona este repositorio
git clone

# Navega al directorio del proyecto
cd reportes-fe

# Instala las dependencias
npm install

# Inicia la aplicación
npm run dev
```
